# Garden Review Bot

Backend‑сервис для сбора отзывов о филиалах из внешних площадок и оперативного уведомления команды через Telegram. Есть закрытая веб‑панель для просмотра, фильтрации, статистики и экспорта.

## Релевантно для PHP‑разработчика
- Laravel 12, PHP 8.2, выделенная доменная логика и UseCase‑слой.
- Очереди задач и фоновые Job‑процессы; мониторинг через Horizon.
- Интеграции с внешними API (Yandex Maps, 2GIS, Telegram) и обработка данных.
- Экспорт данных в Excel (maatwebsite/excel), работа с коллекциями и фильтрами.
- Контроль доступа: auth + токены для публичных URL.

## Что делает проект
- Сбор отзывов из Yandex Maps, 2GIS и Yandex Vendor.
- Уведомления о новых отзывах в Telegram‑боте.
- Веб‑панель для просмотра, фильтрации и детальной карточки отзыва.
- Выгрузка отфильтрованных данных в `.xlsx`.

## Ключевые модули
- Jobs: фоновые сборы и уведомления.
- UseCases: бизнес‑сценарии, статистика, экспорт.
- ReviewController: API и веб‑маршруты для статистики, выгрузки и таблицы.

## Стек
- Backend: Laravel 12, PHP 8.2, Redis (Predis), Horizon, Excel (maatwebsite/excel), Telegram Bot SDK.
- Frontend: Inertia.js + React, Vite, Reactstrap, Tailwind CSS.
